## Overview

Comprehensive educational materials for pediatric neurology, focusing on seizures, epilepsy, and the modern genetic era of diagnosis and treatment. Designed for medical students and residents.

## Contents

| Document | Description |
|----------|-------------|
| [Pediatric Neurology - Seizure](Pediatric_Neurology_Seizure.md) | Complete lecture notes covering seizures, EEG, genetic epilepsies, and precision medicine |

## Topics Covered

- **Seizure Fundamentals** - Definitions, semiology, and classification
- **ILAE Classification** - 2017 framework with 2022/2025 updates
- **Electroencephalography (EEG)** - Age-dependent patterns and interpretation
- **Genetic Epilepsies** - Major genes (SCN1A, KCNQ2, STXBP1, CDKL5)
- **Diagnostic Workup** - First seizure evaluation algorithm
- **Phenotyping** - HPO and standardized terminology
- **Precision Medicine** - Gene-specific treatment strategies
- **Clinical Pearls** - Practical tips and safety considerations

## Key Features

- Evidence-based content with peer-reviewed references
- Interactive self-assessment questions
- Clinical algorithms and flowcharts
- Gene-specific treatment tables
- Emergency medication guidance (especially for Dravet syndrome)
